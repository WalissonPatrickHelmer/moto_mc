🏍️ Moto IA na Estrada M.C.

Website institucional desenvolvido para apresentação digital do Moto IA na Estrada Moto Clube, combinando identidade biker com tecnologia moderna e conceitos de Inteligência Artificial.

O projeto foi criado utilizando HTML5, Tailwind CSS e JavaScript, com foco em responsividade, experiência do usuário e internacionalização dinâmica.

📖 Sobre o Projeto

O site simula a presença online de um moto clube moderno, integrando elementos visuais fortes com funcionalidades interativas, como:

🌍 Sistema multilíngue dinâmico

📱 Layout totalmente responsivo

⚡ Interface moderna com Tailwind CSS

💬 Integração direta com WhatsApp

🧠 Conceito visual inspirado em IA e tecnologia

🎯 Navegação SPA-like (scroll por seções)

A proposta é demonstrar habilidades em desenvolvimento frontend, organização de layout e construção de interfaces modernas orientadas à experiência do usuário.

🚀 Funcionalidades
🧭 Navegação Inteligente

Menu fixo (sticky navbar)

Versão desktop e mobile

Menu hamburguer responsivo

🌐 Internacionalização (i18n)

Sistema de tradução em tempo real com suporte para:

🇧🇷 Português (Brasil)

🇺🇸 Inglês

🇪🇸 Espanhol

🇩🇪 Alemão

🇫🇷 Francês

O conteúdo é alterado dinamicamente via JavaScript sem recarregar a página.
